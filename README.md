## Installation
```
git clone https://github.com/mickaelnambs/porfolio.git
cd portfolio
composer install
npm install
composer prepare
```

## Configuration
Créer un fichier `.env.local` : 
```dotenv
DATABASE_URL=mysql://root:root@127.0.0.1:3306/portfolio
```

## Démarer le serveur
```
symfony serve
npm run dev
```
